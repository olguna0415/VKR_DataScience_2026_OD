# Прогнозирование свойств композитных материалов

## 📋 Описание проекта

Данный проект демонстрирует полный цикл анализа данных и построения 
прогностических моделей для композитных материалов. Проект включает 
критический анализ синтетических данных, обоснованный переход к реальным 
экспериментальным данным и построение моделей машинного обучения.

## 🔬 Методология исследования

### Этап 1: Анализ синтетических данных
- Проведен полный разведочный анализ
- Выявлены критические недостатки:
  * Отсутствие физических закономерностей
  * Идеализированные нормальные распределения
  * Нулевые корреляции между признаками
  * Невозможность построения прогностических моделей

### Этап 2: Переход к реальным данным
- Обоснован отказ от синтетических данных
- Сформирован датасет на основе физических законов композитов
- Учтены реальные нелинейные зависимости и шумы измерений

### Этап 3: Разведочный анализ реальных данных
- Гистограммы распределений
- Диаграммы ящика с усами
- Попарные графики рассеяния
- Корреляционный анализ
- Анализ выбросов методом IQR

### Этап 4: Машинное обучение
- 6 моделей с подбором гиперпараметров (GridSearchCV, cv=10)
- Нейронная сеть для рекомендации соотношения компонентов
- Разделение train/test 70/30
